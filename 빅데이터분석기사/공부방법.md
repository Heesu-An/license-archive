## **팁**


함수명이나 파라미터 등이 기억 안날 때는 아래 사용하면 된다. 사용법 좀 익혀야 할 듯.

몇 개 없는데 쓸 일이 있을까 싶었는데, 몰랐던 것들 나왔을 때나 실제로 기억 안나서 아래 함수 많이 사용했다.

```
print(help(df.dropna))   # 함수: 파라미터 기억 안날 때
print(dir(sklearn.preprocessing)    # 라이브러리: 함수명 기억 안날 때
```
