# **\[ 검정 개념 정리 \]**

1\. F-검정: 두 집단의 '분산'이 같은지 확인

-   **F-통계량 ≥ 1** ⇒ **분자 > 분모**
-   ex. 회사1과 회사2의 직원 연봉 분산 비교
-   양측검정
-   p-value < 0.05 ⇒ 두 집단의 분산 다름 (대립가설 채택)
-   p-value ≥ 0.05 ⇒ 두 집단의 분산 같음 (귀무가설 채택)

3\. T-검정: 두 집단의 \*\*‘평균’\*\*이 같은지 확인

-   표본 구분
    -   독립표본 T-검정: 두 집단 간 평균 ex. 두 약의 효과 비교
    -   대응표본 T-검정: 같은 사람의 전 후 비교 ex. 복용 전후 체중 변화
-   양측검정: 평균이 다르다 (방향없음)
-   단측검정: 한쪽이 크거나 작다 (오른쪽은 크냐, 왼쪽은 작냐)
-   p-value < 0.05 ⇒ 두 집단 평균 차이 유의 (대립가설 채택)
-   p-value ≥ 0.05 ⇒ 두 집단 평균 차이 유의 안함 (귀무가설 채택)

## **\[ (🚨🚨🚨🚨내 생각, 제일 중요) 검정 흐름 \]**

**<수치형 변수 분석 절차>**

**1\. t-검정 (두 그룹 평균 비교)**

**귀무가설(H0): 두 그룹의 평균은 차이가 없다.**



<br>



✔ **언제 사용?**

-   그룹이 두 개(예: 약 복용 vs 비복용)
-   종속변수가 **연속형(수치형)**
-   **정규성 만족해야 한다** (두 그룹 각각)
-   **등분산 여부 검사 후 진행**



<br>


✔ **종류별 코드**

```
# 독립표본: 서로 다른 두 그룹   ex. 약 복용 그룹과 복용하지 않은 그룹
ttest_ind(a, b, equal_var=True/False)

# 대응표본: 같은 그룹의 전·후 비교  ex. 약 복용 전 후
ttest_rel(a, b)

# 단일표본: 한 그룹 평균이 특정 값과 같은지
ttest_1samp(a, mean)​
```


<br>


✔ **등분산에 따른 파라미터 조정 (등분산 검정 후)**

-   등분산이면 equal\_var=True
-   등분산 아니면 equal\_var=False



<br>




**2\. ANOVA (분산 분석)**

**귀무가설(H0): 여러 그룹의 평균은 모두 같다.**

✔ **언제 사용?**

-   그룹이 **3개 이상**
-   종속변수가 수치형
-   그룹 간 평균 차이 전체를 먼저 검정
-   정규성 + 등분산 필요
    -   **등분산 검정 필요함 (Levene) → 보통 ANOVA 전에 수행함**


<br>


✔ **코드**

```
stats.f_oneway(group1, group2, group3, ...)
```


<br>


**3\. 정규성 검정**

**귀무가설(H0): 정규성을 만족한다.**

**✔ 기각 기준 (다른 검정이랑 다름!!!)**

-   **p-value > 0.05 → 정규성 O (귀무가설 기각하지 않음)**
-   **p-value ≤ 0.05 → 정규성 만족 X (비모수 검정으로 이동)**

✔ 표본 크기에 따른 종류 및 코드

```
# 표본 5000이하
stats.shapiro(그룹1)  # 그룹별로 검정

# 표본 5000이상
stats.anderson(그룹1)
stats.kstest(그룹1)​
```


<br>


**4\. 등분산 검정**

**귀무가설(H0): 두 그룹의 분산은 같다. (등분산이다)**

✔ p > 0.05 → 등분산 O → t**\-test에서 equal\_var=True**

✔ 종류 및 코드

```
stats.levene(a, b)
stats.bartlett(a, b)
stats.fligner(a, b)
```


<br>


**5\. 비모수 검정**

: **정규성 불만족 시** 대체로 사용하는 검정.

| **표본** | **이름** | **코드** |
| --- | --- | --- |
| **독립표본** | 맨-휘트니 U검정 (중위수 비교) | **stats.mannwhitneyu(a, b)** |
| **독립표본** | 윌콕슨 중위수 | **stats.ranksums(a, b)** |
| **대응표본** | 윌콕슨 순위합 | **stats.wilcoxon(a, b)** |


<br>


#### **<범주형 변수 분석 절차>**

**귀무가설(H0): 실제 분포는 기대 분포와 차이가 없다.**

pooled_var = ((n1-1)*(분산1) + (n2-1)*(분산2)) / (n1 + n2 - 2)  # 합동 분산 추정랭
T = (mean(g1)-mean(g2)) / np.sqrt(pooled_var(1/n1 + 1/n2)))   # T-통계량(평균 같은지 검정)
CDF = stats.t.cdt(abs(T), (n1+n2-2))   # T-검정에서의 CDF(누적분포함수) : stats.t.cdt(검정통계량, 자유도)
p-value = 2*(1-CDF)               # p-value

| **평가지표** | **코드** |

| **MSE** | mean\_squared\_error(y\_true, y\_pred) |

3️⃣ 분류 평가 지표

F1-score가 평가지표로 지정되어서 시험에 나왔음. average='macro' 로 설정했어야 함. help 통해서 해결했음.

| **Recall / Sensitivity (재현율, 민감도)** | recall\_score(y\_true, y\_pred) | average='binary' \| 'macro' \| 'weighted' |
| **Specificity (특이도)** | recall\_score(y\_true, y\_pred, pos\_label=0) | pos\_label=0 으로 음성 클래스 기준 재현율 |
| **F1-score** | f1\_score(y\_true, y\_pred) | average='binary' \| 'macro' \| 'weighted' |
| **Accuracy** | accuracy\_score(y\_true, y\_pred) |   |
| **ROC-AUC** | roc\_auc\_score(y\_true, y\_prob) |   |
