    -   이상값에 민감 → 이상값 처리, k-median 등으로 보완
-   혼합 분포 군집: 모형 기반 군집 방법. EM 알고리즘 사용
    -   E-단계: Z 기대치 계산
    -   M-단계: Z 기대치 기반 파라미터 추정

    -   **순전파** 사용으로 속도 매우 빠름

**<연관분석>**

-   연관 분석 (비지도)
    -   연관규칙분석(=장바구니 분석)
    -   순차분석
-   🚨 연관 분석 지표
    
    -   A→B일 때
    
    -   **지지도 = 𝑃(𝐴 ∩ 𝐵) / (전체거래수)**
    -   **신뢰도 = 𝑃(𝐴 ∩ 𝐵) / P(A)**
    -   **향상도 = {𝑃(𝐴 ∩ 𝐵) \* (전체거래수)} / {P(A) \* P(B)}**
        -   **향상도 = 1: 서로 독립 관계**
        -   **향상도 > 1: A를 구매 후 B 구매할 확률 높음**
        -   **향상도 < 1: 연관 없음**
